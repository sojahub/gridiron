# Querying via CLI
 
<Callout type="info" emoji="ℹ️">
  Gridiron Ethereum CLI will work the same way as Cosmos SDK. Replace `simd` with `grid`
</Callout>

Gridiron Ethereum does not extend the use of CLI querying past the base implementation of the Cosmos SDK.
For information on querying via CLI in Gridiron Ethereum you can reference [Using the CLI](https://docs.cosmos.network/main/run-node/interact-node#using-the-cli) 
from the Cosmos SDK documentation.
